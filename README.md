# My dotfiles

Managed by [rcm].

## Installation

1. Clone this repo

```
git clone https://github.com/yuriihabrusiev/dotfiles.git ~/.dotfiles
```

2. Install [rcm]

```
brew tap thoughtbot/formulae
brew install rcm
```

3. Install the dotfiles

```
env RCRC=$HOME/.dotfiles/rcrc rcup
```

[rcm]: https://github.com/thoughtbot/rcm
